How to Use My Plugin

Welcome to My Plugin! This plugin helps you automatically generate supporting classes for entities in your project. It is compatible with java (preferably 17) and Kotlin

Quick Start

  1. When you create a new entity in your project, a prompt will appear asking if you would like to generate a Repository, Service, Controller, Dtos, Mapper class for it.
  2. If you agree, the plugin will automatically generate the corresponding classes in their respective folders (repository, service, controller, mapper, dtos).

Configuration

To specify which packages should trigger this generation feature, you have two options:

  1. Tools > Set Entity Packages - that's where you should input names of the packages (if there is more than one package you want to configure, please use ",")
  2. File > Settings > My Plugin Settings and just follow the written

Note: if you do not have packages named "repository", "service", "controller", "mapper" and "dtos" with those exact names the plugin will make them for you, and put newly generated classes and/or interfaces in them

Also, make sure that when you specify packages for classes generation, it should be from java/kotlin package, for example: com.example.mypackage
